3.3.5. Radio

La tarjeta micro:bit puede comunicarse de forma inalámbrica con otras micro:bits usando radio. Esta tecnología inalámbrica es útil para poder enviar mensajes entre distintas tarjetas y cuando no se pueda conectar el sensor con la unidad receptora de datos mediante cable. Vamos a ver qué instrucciones podemos usar para manejar la radio en la tarjeta micro:bit que, evidentemente, están en el bloque radio. Podemos clasificarlas en tres categorías:

  • Grupo
  • Enviar
  • Recibir

Tarea

Ejercicio 1

En este programa trabaremos juntos dos equipos. Se trata de crear un pato que viajará mágicamente a través del aire de una tarjeta micro:bit a otra. Para enviarlo de vuelta, solo habrá que agitarlo.

Se deben seguir los siguientes pasos:

  • La profesora os indicará qué dos grupos trabajaréis juntos
  • Cada uno de los grupos creará su propio programa. En ambos se deberá establecer el mismo grupo de radio (os lo asignará vuestra profesora)
  • El programa se iniciará cuando la tarjeta micro:bit se agite. En ese momento, se borrará la pantalla y se enviará un mensaje con la palabra "pato" a la otra tarjeta
  • Cuando la otra tarjeta reciba el mensaje, mostrará el icono del pato

Voluntario

Crearemos un programa en el que se enviarán preguntas de una tarjeta micro:bit a otra, a las que se responderá con sí o no. Procedimiento:

  • Establecer el mismo grupo de comunicación para los dos equipos
  • Implementa las instrucciones necesarias para que al pulsar el botón A se envía "sí" y al pulsar el botón B se envíe "no". Justo después de la instrucción de enviar, se mostrará el icono de acierto o de fallo, pararemos medio segundo (500ms) y se borrará la pantalla
  • Cuando se reciba el mensaje, la tarjeta mostrará el icono correspondiente al sí (un aspa) si se ha recibido ese mensaje o el del no (una cruz), si se ha recibido un mensaje "no". El icono se mostrará medio segundo y después se borrará la pantalla
